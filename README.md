# Academic Projects

Репозиторий содержит **учебные проекты**, выполненные в рамках университетской программы.  
Представленные работы демонстрируют **практический опыт** в областях:

*   **Системного программирования**
*   **Распределённых вычислений**
*   **Контейнеризации**
*   **Основ веб-разработки**

---

## Технологии и навыки

| Категория | Технологии |
|-----------|------------|
| **Языки программирования** | C, C++, Python, JavaScript, Bash |
| **Инфраструктура и DevOps** | Docker, Kubernetes, GitLab CI |
| **Системное администрирование** | Linux, модули ядра, сборка систем |
| **Параллельные вычисления** | OpenMPI, кластерные системы |
| **Веб-технологии** | HTML, CSS, JavaScript |

---

## Структура проектов

### **Алгоритмы и структуры данных**
*   **`decimal`** — реализация арифметики с **высокой точностью**
*   **`matrices`** — работа с матричными операциями и алгоритмами

### **Университетские лабораторные работы**
*   **`parallel/`** — параллельное программирование (**OpenMPI**), работа с распределёнными системами
*   **`cpp/`** — проекты на **C/C++** (консольные приложения, алгоритмы)
*   **`web/`** — веб-приложение для учёта бюджета на **JavaScript**
*   **`infra/`** — инфраструктурные задачи:
    *   Развёртывание сервиса с использованием **Docker** и **Kubernetes**
    *   Написание модулей для ядра **Linux**
    *   Сборка ядра **Linux**

Проекты демонстрируют способность работать с **разноуровневыми технологиями** — от низкоуровневого программирования на C до оркестрации контейнеров. Каждая работа представляет собой **законченное решение** учебной задачи с акцентом на понимание фундаментальных принципов.
